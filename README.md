# 🤖 J.A.R.V.I.S. (Just A Rather Very Intelligent System)

An advanced AI-powered desktop voice assistant inspired by Marvel's J.A.R.V.I.S., built completely in Python. It combines voice recognition, desktop automation, a futuristic GUI, AI-powered screen understanding, and intelligent system controls to create a real personal assistant experience.

---

# ✨ Features

## 🎤 Voice Assistant
- Wake and interact using natural voice commands.
- Continuous microphone listening.
- High-quality Microsoft Edge TTS voice.
- Startup sound effects.
- Intelligent command recognition.

---

## 🖥️ Futuristic GUI
- Modern J.A.R.V.I.S.-style interface.
- Animated central holographic sphere.
- Floating Button
- Live digital clock and date.
- CPU usage monitor.
- RAM usage monitor.
- Battery percentage display.
- Wi-Fi signal strength indicator.
- Power status indicator.
- Live microphone status.
- Dynamic response panel.
- Status updates such as:
  - Preparing Screen...
  - Analyzing...
  - ✓ Screen Analyzed
  - ✓ Error Analyzed

---

## 📸 AI Screen Analysis
Analyze the entire screen using Google Gemini Vision.

Commands:
- Analyze my screen
- Analyze screen
- What's on my screen
- Explain my screen

Features:
- Automatically captures a screenshot.
- Understands the complete screen context.
- Gives concise and natural explanations.
- Five-second preparation timer before capture.
- Custom scanning sound effect while analyzing.

---

## 🐞 AI Code Error Analysis
Solve programming errors directly from the screen.

Commands:
- Solve this error
- Explain this error
- Analyze this error

Features:
- Detects visible error messages.
- Explains the error in simple language.
- Suggests possible solutions.
- Displays analysis status in the GUI.
- Gracefully handles Gemini API quota errors.
